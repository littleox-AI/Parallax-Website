視差滾動效果

作者：Vineet Kumar

什麼是視差滾動？

視差滾動是一種網頁設計技巧，讓網站背景比前景移動得更慢。訪客向下捲動時會產生 3D 效果， 增加深度並帶來更沈浸的瀏覽體驗。視差基於視覺錯覺。由於人眼感知較近的物體比遠處的物體更 大，我們會覺得遠處的物體移動得更慢。這種錯覺長期被不同媒介的視差效果採用，營造真實感。 最早可追溯到傳統動畫，如迪士尼《白雪公主與七矮人》，以及像《超級瑪利歐》這類電玩。隨著 CSS 與 HTML 的進步，視差效果也演變成今日熟悉的網頁設計。

其中一個最早的瀏覽器視差滾動實作由開發者 Glutnix 於 2007 年在部落格分享，其中包含範例程 式碼與使用 JavaScript 和 CSS2 的示範，支援當時的 IE6 等瀏覽器。不過直到 2011 年並引入 HTML5 與 CSS3 後，這項技巧才在網頁設計師間流行。支持者認為這是擁抱網路流動性的簡單方式，他們 利用視差背景更好地吸引使用者並提升網站的整體體驗。然而，普渡大學在 2013 年發表的研究指 出：「……儘管視差滾動提升了使用者體驗的某些面向，但並未必然改善整體的使用者體驗。」


為什麼要使用視差圖像？

    設計師應嘗試視差圖像有兩個主要原因：

  • 它有助於講故事

    視差圖像非常適合說故事。好的設計就是溝通——在設計網站時，我們希望與訪客交流，並講 述關於產品、服務或品牌的吸引人故事。視差滾動能讓網站訪客沉浸在獨特的線上體驗中，甚 至能打造單頁網站，讓訪客不用跳到其他頁面，就能讀完你對產品或品牌的完整故事。

  • 有助於提升使用者參與度

    視差圖像可用來提升使用者參與度。降低網站跳出率是網頁設計必須留意的目標之一。跳出率 衡量的是訪客只看了一頁就按下「上一頁」或關閉分頁的比例。過高的跳出率會傷害搜尋排名， 因此任何能降低跳出率的做法都值得採用。精心打造的視差頁面能帶來動態體驗，讓使用者更 願意停留。


如果想在網站加入視差滾動效果，可考慮以下建議。

  • 選擇性使用

    視差滾動在應用於特定元素時效果最佳，而不是每個頁面都套用。可以用在頁首或標題，或只 用於首頁。你希望視差滾動強化網站設計，而非分散使用者注意力。

  • 圖像壓縮

    視差滾動會使用各種媒體檔案與 CSS 形狀來營造深度。你不希望過大的圖像檔拖慢載入速度， 進而影響網站效能與使用者體驗。

  • 善用色彩

    視差滾動不僅與前景、背景圖像有關，也與色彩息息相關。背景可使用較低飽和度的色調，前 景搭配更鮮明的顏色，營造更強烈的深度感。
